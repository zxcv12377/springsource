유효성 검증(Form Validation)
 - 사용자가 입력한 데이터에 대해서 사전 검증하는 작업
 - 입력값을 받는 컨트롤러 단에서 처리
 - 자주 사용하는 유효성에 대한 검사를 도와주는 프레임워크(hibernate validator) 사용 => 어노테이션을 통해 검증 가능

 - 어노테이션 종류
  1) @NotBlank : null, 공백 , 띄어쓰기 모두 안됨
  2) @NotEmpty : 길이가 0인지 검사, (공백허용 => 사용자가 공백으로 문자를 채우면 검증 불가)
  3) @NotNull : Null 체크
  4) @Length : 입력문자 길이 제한
  5) @Email : 이메일 형식에 맞는지 검사(입력값을 입력해야 검사해줌)
  6) @Min, @Max : 최솟값, 최댓값 검사
  7) @Size : 입력 범위 지정
  8) @Pattern : 정규식을 통한 조건
